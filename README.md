# veo3free

这是一款基于google flow的批量生成工具。

## 支持的功能

1. 批量veo3.1视频生成，包括文生视频、首尾帧视频和多参考图视频，分辨率支持1080p和720p
2. 批量nano banana pro图片生成，分辨率支持2k和4k

## 使用步骤
1. 安装和启动本程序
2. 浏览器安装辅助脚本（基于油猴插件，故请先配置油猴），https://greasyfork.org/zh-CN/scripts/561650-flow2ui
3. 浏览器打开 https://labs.google/fx/tools/flow 登录有flow权限的google账号

然后就可以开始使用了
